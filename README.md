# Piano-Meetup-Node
A platform for Piano and Keyboard enthusiast.

Tech stack used:
- Node.js, Express for backend
- EJS, html, css for frontend
- Mongodb, Mongoose for DB.

------------------------------------------------------------------------------------------------------------------------------

To setup:
- npm install

To setup db:
- Install mongodb
- Use scripts in create_db_Final.txt

To run:
- node app
